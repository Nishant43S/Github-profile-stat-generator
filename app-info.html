GitHub Profile Statistics Generator

This app is a simple tool designed to help you showcase your GitHub profile statistics. It's built using Streamlit, which makes it fast and easy to use. Here's what you can expect from the app:

Features

  • Profile Visitor Count Badge: This feature shows how many people have visited your GitHub profile.
  • Profile Statistics: Get insights into your GitHub activities, such as the number of repositories, followers, and contributions.
  • Language Statistics: See which programming languages you use the most in your projects.
  • Profile Streak: Track how many consecutive days you've contributed to your GitHub projects.
  • Repository Statistics Widget: This widget provides detailed information about your repositories.

Customization Options

  • Customizable Widgets: You can change the look of the widgets to match your style.
  • Themes: Choose from different themes like Dracula, Atom One, and Tokyo Night Dark to personalize your profile.
  • Borders: You can enable or disable borders around the widgets to make them look just how you want.

Easy to Use

  1. Fast and Optimized: The app runs quickly, ensuring you get your statistics in no time.
  2. Copy and Paste: Once your statistics are generated, you can easily copy the README code and paste it into your GitHub profile.

How to Use

  1. Enter your GitHub username in the app.
  2. The app will generate your profile statistics.
  3. Customize the widgets and theme as you like.
  4. Copy the generated code and paste it into your GitHub profile.

This app is perfect for anyone looking to enhance their GitHub profile with eye-catching statistics. Enjoy showcasing your coding journey!
